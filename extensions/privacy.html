DevOps Test UI — Privacy Policy

Effective date: September 23, 2025

1. Summary

DevOps Test UI is a Safari extension that records page interactions to create functional tests and uploads those tests to a DevOps Test Hub server you configure. This policy explains what data is collected, how it is used, and your rights.

2. Data collection

Automatically collected during recording (only when you start a recording):

  • Page URL and page title.
  • DOM snapshots and hierarchy (HTML of the recorded page or portions of it).
  • User actions on the page (clicks, form inputs, navigation events, timing info).
  • Screenshots or visual snapshots.

Not collected

  • Browser history beyond the recorded session URLs.
  • Stored passwords or secret fields.

3. Purpose and use

  • Data is used only to build a functional test artifact (script + assets).
  • Test artifacts are uploaded to your configured DevOps Test Hub server.
  • Minimal diagnostic information may be collected to troubleshoot failed uploads (if enabled).
  • No recorded DOM or screenshot data is sent to analytics services.

4. Data Sharing & Third Parties

  • Test artifacts are uploaded to the DevOps Test Hub server you configure and are accessible under that server’s access controls.
  • We do not sell or lease your data.
  • Diagnostic data (if enabled) may be sent to crash/analytics services (e.g., Apple crash reporting, Sentry). These reports do not include DOM or screenshots.

5. Retention

  • Uploaded tests are stored on your DevOps Test Hub server and subject to that server’s retention policies.
  • Local cached recordings (if any) are retained only until upload succeeds.
  • You may clear cached recordings manually in the extension settings.

6. Security

  • Data in transit is encrypted using HTTPS/TLS.
  • The extension stores minimal state and does not persist sensitive values by default.
  • Ensure your DevOps Test Hub server has a valid TLS certificate.

7. User controls & choices

  • You control the DevOps Test Hub endpoint to which tests are uploaded.
  • Delete local cached recordings from extension settings.
  • To delete artifacts on the server, contact your DevOps Test Hub administrator or use the server UI/API.

8. Children

This extension is not directed at children under 13 (United States, COPPA). In the European Union, this extension is not directed at children under 16. We do not knowingly collect personal data from children.

9. Changes to this policy

We may update this policy. Updated policies will be posted inside the extension's About page with an effective date.
